Puede ser que podría ser más desafiante, se podría hacer que ciertos niveles tengan una grilla mas grande, o se podría diseñar mejor los niveles. Lo de agrandar las grillas no sería tan dificil, ya que no esta tan hardcodeado en el juego, pero la verdad no creo que llegue a hacerlo. 
Pienso que la mecánica base es simple pero da la suficiente flexibilidad como para hacer desde algo super facil a algo super difícil. No me arrepiento de la mecánica que elegí.

Pensé en agregar mas cosas como pistas, algun bloque especial, monedas que hay que agarrar en el nivel, etc. Pero lo evalué y me pareció fuera del enfoque del proyecto, tal vez en un futuro lo agregue.

Para el agua use de base un proyecto ya existente, a partir de este código lo fui modificando a lo que necesitaba. El mayor cambio fue pasar de physicsserver a nodos "reales". También fui editando diversos parámetros para aumentar la fluidez del agua entre otras cosas.

Otro cambio que hize fue cambiar como se muestra el agua. originalmente tenía una especie de shader, el cual no estaba muy bien hecho y decidí usar puntos azules para cada gota, que le da un toque caricaturezco y lo hace mas facil de manejar.

Puede que me haya esmerado un poco mucho en el diseño gráfico :P
